# clasePracticandoGitHub

## Requerimientos Funcionales
 * El usuario puede escribir su prompt y debe recibir una respuesta
 * El sistema debe mantener el contexto de anteriores chats
 * El sistema debe mantener un registro de los anteriores chats

## Requerimientos No funcionales
  * El sistema debe responder en menos de 5 segundos
  * El usuario tiene un limite de 50 mensajes o prompts diarios
  * El sistema almacena un limite de 100 registros de diferentes chats
